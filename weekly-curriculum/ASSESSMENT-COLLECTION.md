# Assessment Collection

## Preferred Method

Students complete the HTML assessment and click `Download Results (CSV)` on the final results screen.

Collect the CSV files using:

1. Google Drive upload folder
2. Email
3. Shared network folder or LMS upload
4. Flash drive, only as a fallback

See `GOOGLE-DRIVE-COLLECTION.md` for the recommended Google Drive folder structure.

## File Names

The assessments create files like:

```text
vub-pre-test-jane-smith-2026-04-27.csv
vub-post-test-jane-smith-2026-06-01.csv
```

## Instructor Notes

- The pre-test and post-test still support `Print Report (PDF)`.
- The post-test can compare against the pre-test only if the student uses the same browser profile/computer or if local storage is preserved.
- The CSV file is the official collection artifact because it works even when students are on separate machines.
