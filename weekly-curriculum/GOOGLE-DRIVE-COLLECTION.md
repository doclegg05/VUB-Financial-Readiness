# Google Drive Assessment Collection

Use Google Drive as the main place where students return their pre-test and post-test CSV files.

## Recommended Drive Folder Structure

Create one Google Drive folder named:

```text
VUB Financial Readiness - Assessment Results - Spring 2026
```

Inside it, create:

```text
01 Pre-Test Results
02 Post-Test Results
03 Combined Results
04 Backup / Late Submissions
```

## Sharing Settings

Use the safest sharing option available in your setting:

```text
Anyone with the link can upload/edit
```

If your school or organization restricts uploads, use:

```text
Anyone in the organization with the link can upload/edit
```

If neither option is available, use email collection as the backup.

## Results Tracker

This course includes a starter CSV template:

```text
templates/vub-assessment-results-tracker.csv
```

Upload that file to Google Drive and open it with Google Sheets. It gives you the columns that match the CSV files students download from the pre-test and post-test.

Suggested Google Sheet name:

```text
VUB Financial Readiness - Assessment Results Tracker - Spring 2026
```

## Student Workflow

For Week 1:

1. Complete the pre-test.
2. Click `Download Results (CSV)`.
3. Upload the CSV file to the `01 Pre-Test Results` Google Drive folder.
4. File name should look like `vub-pre-test-last-name-2026-04-27.csv`.

For Week 6:

1. Complete the post-test.
2. Click `Download Results (CSV)`.
3. Upload the CSV file to the `02 Post-Test Results` Google Drive folder.
4. File name should look like `vub-post-test-last-name-2026-06-01.csv`.

## Instructor Verification

Before students leave class:

1. Open the Drive folder.
2. Confirm one CSV file appears for each student.
3. Ask any missing student to re-upload or email the file before leaving.

## Backup Plan

If Google Drive uploads fail:

1. Students email the CSV file to you.
2. Subject line:

```text
VUB Pre-Test Results - Last Name
VUB Post-Test Results - Last Name
```

3. If email also fails, collect files with a flash drive.
