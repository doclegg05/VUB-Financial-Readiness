# Verify Benefit Information

Audit all financial and benefit claims in the course content for accuracy.

For each module, check:
1. **Eligibility rules** — Are stated requirements current?
2. **Dollar amounts** — Are any specific amounts outdated?
3. **Program names** — Have any programs been renamed or discontinued?
4. **Regulations** — Are cited rules still in effect?
5. **Deadlines/windows** — Are timeframes (e.g., BDD 90-180 day window) accurate?

Cross-reference against official sources:
- VA.gov
- OPM.gov (CSRS/FERS)
- DFAS.mil (military pay)
- SSA.gov (Social Security)
- Medicare.gov
- TRICARE.mil

Output findings as a checklist with status (Verified / Needs Update / Unable to Verify).
